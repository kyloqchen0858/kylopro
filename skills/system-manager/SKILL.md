# System Manager

## 何时使用

当用户提到以下主题时使用：

- 磁盘空间
- 软件清理
- 卸载不用的工具
- 查看系统里装了什么
- 想知道哪些软件已经可以被 Kylopro 替代

## 如何执行

优先使用安全、可审计的 PowerShell 或命令行方式：

1. 先扫描，不要直接卸载
2. 先输出摘要，再等待确认
3. 涉及删除、卸载、系统改动时，必须获得用户当前消息中的明确确认

## 推荐执行路径

- 软件列表：PowerShell / winget / 系统查询命令
- 磁盘占用：PowerShell + 文件系统工具
- 建议而非直接动作：先给出可删除项、风险和收益

## 重要规则

- 永远不要在没有明确确认的情况下执行卸载
- 优先做分析报告，不要先做破坏性操作
- 如果现有 nanobot 工具足够，就不要再造专用系统管理框架
